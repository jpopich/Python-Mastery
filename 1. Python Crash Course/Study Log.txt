-----CHAPTER 4: WORKING WITH LISTS-----
For each loop over elemsnts of a list.
range() - range(inclusive, exclusive)
min() - returns minimum of list
max() - returns maximum of list
sum() - sums list total

List comprehension template: 
arrVar = [expression << forLoop]
variableToStoreList = expression we want to feed the loop into.
example: squares = [values**2 for value in range(2, 11)]

Slicing
list[inclusive:exclusive]
print(players[0:4]) -> only players at 0, 1, 2, and 3 are printed.
list[:3] - begins at first index up to 3 exclusive
list[3:] - begins at 3 inclusive to final index
list[-3:] - last three elements of list
list[-6:-1:2] - last six elements of list skipping by 2


Copying a list:
newList = oldList[:] - start at first and end at last index
newList = oldList - they are both referencing same list in memory
